jQuery Libraries

Search on the internet for some cool jQuery libraries. http://tutorialzine.com/2013/04/50-amazing-jquery-plugins/ or http://designshack.net/articles/javascript/40-awes... could be a good start although you can just google as well. When you're building your project, it's helpful that you understand what other jQuery/JavaScript libraries are out there. Get familiar with some cool JavaScript libraries and see if you can make something cool! Later in the bootcamp, you'll learn more about JavaScript and you'll know how to use more advanced JavaScript libraries. For now, try to get yourself familiar with some cool JavaScript libraries as well as what these will allow you to do. We're seeing that more companies are looking for developers who understand how to use JavaScript as well as those who know how to quickly integrate other JavaScript libraries. This is your chance to get into the world of JavaScript. :)

Fit Text

Fittext.js is a jQuery plugin for inflating web type. It automatically scales the text of an element to take up the full width of its container. This makes the layout responsive and looking sharp on any device.
